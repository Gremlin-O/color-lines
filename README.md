# Color Lines

**Color Lines** — это современная реализация классической браузерной игры «Шары в ряд», разработанная с использованием актуального frontend-стека. Проект демонстрирует навыки построения интерактивных интерфейсов, управления состоянием и организации масштабируемой архитектуры.


---

## Описание проекта

Цель игры — набирать очки, выстраивая линии из шаров одного цвета. Игрок перемещает шары по игровому полю, стараясь собрать линию из 5 и более одинаковых элементов по горизонтали, вертикали или диагонали.

После каждого хода на поле появляются новые шары, что постепенно усложняет игру и требует стратегического мышления.

---

## Основные механики

- Перемещение шаров по свободным клеткам (с проверкой пути)
- Генерация случайных шаров после хода
- Определение линий (горизонталь, вертикаль, диагонали)
- Удаление собранных линий
- Подсчёт очков
- Проверка проигрыша (когда поле заполнено)

---

## Технологический стек

- **Vite** — быстрый сборщик и dev-сервер  
- **React** — построение UI  
- **TypeScript** — строгая типизация  
- **Redux Toolkit** — управление состоянием
